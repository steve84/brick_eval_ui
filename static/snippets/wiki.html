Begriffe aus der Lego-Welt

In diesem Kapitel werden einige geläufige Begriffe erläutert.

Steine

Lego-Steine existieren in verschiedenen (Guss-)Formen und Farben. Jede Form erhält eine eindeutige Nummer (Steine-Id), zusammen mit der Farbe (Farb-Id) entsteht somit ein Element (Element-Id), mit welcher jeder einzelne Stein exakt identifiziert werden kann. Die Häufigkeit einzelner Formen und Farben sind sehr unterschiedlich und können auf dem Zweitmarkt durch eine hohe Nachfrage enorm im Preis steigen.

Set

Eine Ansammlung von Steinen, welche in einer Umverpackung (Karton, Plastik- oder Papiertüte) zusammengefasst werden. Jedes Set hat eine eindeutige Nummer (aktuell bis maximal 5 Ziffern), welche oben links aufgedruckt ist. Ein Set wird normalerweise nur ein Mal aufgelegt - Ausnahmen bilden zum Beispiel die Saturn V Rakete (Set-Nummer 21309 im Jahre 2017 und 92176 in 2020) sowie das Schiff in der Flasche (Set-Nummer 21313 im Jahre 2018 und 92177 in 2020), welche praktisch in gleicher Zusammenstellung neu aufgelegt wurden. Zu einem Set gehören auch ein Name sowie eine Altersempfehlung. Ein Set kann keine, eine oder mehrere Minifiguren enthalten. Diese können auch mehrfach darin vorkommen.

Minifigur

Eine Minifigur besteht in der Regel aus 4 Teilen:
einem Kopf
einem Torso
zwei Armen ohne Hände
zwei Hände
Beine
Kopfbedeckung (Haare, Helm oder Hut)
Der Torso, die Arme und die Hände werden jedoch immer zusammengesteckt in den Sets verpackt. Die beiden Arm- sowie Handpaar müssen nicht immer aus zwei identischen Teilen bestehen.

Minidoll

Eine Minidoll ist grösser als eine reguläre Minifigur und besteht aus anderen (Guss-)Formen. Wie es der Name schon sagt, stellt sie eher eine Puppe als eine Männchen dar. Daher erscheinen solche Puppen auch in Sets, welche eher die weibliche Kundschaft ansprechen soll.

Verfügbarkeit / EOL

Ein Lego-Set durchläuft folgenden Lebenszyklus:
Reguläre Produktion durch Lego
Produkt wird von Lego im Shop als Letzte Chance beziehungsweise Einstellung in Kürze markiert
Produktion wird eingestellt (End of Life, kurz EOL), das Produkt ist jedoch mehrere Monate/Jahre im Detailhandel erhältlich (bis die Lagerbestände aufgebraucht sind)

Es kann auch vorkommen, dass Lego die Produktion von Sets einstellt, bei welchen der EOL Status nicht angekündigt wurde. Anhang von Erfahrungswerten kann das Produktionsende eines Sets vorhergesagt werden (Status EOL erwartet)
